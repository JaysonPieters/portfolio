Don't dabble in darkness

Dont dabble in darkness is a semi-horror game where the player must reach a correct door while trying to evade a monster chasing them.

Here are some of the key feature.

Optimized navmesh calculations

While making the game, I eventually noticed the map became so large the navmesh calculation took longer to process than it actually moves. To combat this i made a small piece of code that worked wonders.

More Info

Each run different

Because it is quite a short game, I wanted it to be replayable. To do this I made it so almost everything is randomized. This includes player spawn, monster spawn and the correct doors.

More Info

Project Info

This project was made in 4 weeks, by myself. Due to the simplicity of the game I wanted to do something extra, I wanted to make the game actually look like a game. I used some assets from the unity store, and a few tutorials to make the lighting and fog. All of this made the game look quite good. Most of the time was spent level designing and making the lighting.
